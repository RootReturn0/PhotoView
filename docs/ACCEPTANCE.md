# PhotoView 功能验收清单

## 验收原则

- [ ] 本清单覆盖 `plan.md` 中的全部功能需求、非功能需求和用户场景。
- [ ] 阶段划分只控制实现顺序，不裁剪 `plan.md` 功能；任何阶段完成都不代表未实现功能被移出范围。
- [ ] 所有功能必须在 Tauri 跨平台桌面应用中完成，核心数据和图片资产默认本地处理。
- [ ] 每个功能完成时必须能通过人工验收或自动化测试验证，并记录已覆盖的平台、测试数据和限制条件。
- [ ] 涉及删除、移动、合并、拆分、恢复等高风险操作时，必须有确认流程、失败提示和可追踪结果。

## 前置依赖

- [ ] 数据库已建立 collections、images、tags、favorites、history、settings 等核心数据表；重启应用后合集、图片、标签、收藏、最近浏览和设置数据仍可读取。
- [ ] schema migration 可从空库初始化，也可在已有旧库上升级；升级失败时不会破坏原数据库文件。
- [ ] 文件夹导入能力可选择本地目录，扫描支持格式图片，并把合集、图片路径、格式、大小、时间、分辨率等元数据写入数据库。
- [ ] 增量扫描可重复执行；同一路径图片不会重复入库，新增、删除、重命名文件会产生可验证的数据库变化。
- [ ] 缩略图生成、缓存、命中、失败重试和清理接口已完成；合集网格、图片网格、搜索结果、收藏页依赖该能力显示预览图。
- [ ] 图片解码能力覆盖 jpg/jpeg、png、gif、bmp、ico、tiff、webp、avif、svg；解码失败时界面显示错误状态而不是崩溃。
- [ ] Rust command/API、前端状态、缓存目录、配置目录和文件系统权限边界清晰；前端不能直接越权访问未授权路径。
- [ ] 导入、数据库、缩略图是图片浏览、合集管理、搜索、收藏/最近、重复检测和文件监听的前置依赖，未完成前不得将这些上层功能标记为完成。

## 图片查看器

- [ ] 格式支持：使用测试集分别打开 jpg/jpeg、png、gif、bmp、ico、tiff、webp、avif、svg，均能显示图片或动图首帧/动图播放策略，并在信息面板显示正确格式。
- [ ] 大图加载：首次打开 4K+ 图片不超过 1 秒，已缓存再次打开不超过 200ms；加载期间有明确占位或进度状态。
- [ ] 缩放：支持适应窗口、实际大小、滚轮自由缩放和快捷键缩放；切换模式后图片比例正确，缩放状态在当前查看会话内可见。
- [ ] 旋转：支持 90° 旋转；连续旋转四次后回到原始方向，且不修改原文件。
- [ ] 翻页：键盘左右键、鼠标侧键和界面按钮都能上一张/下一张；到达首尾图片时有边界处理。
- [ ] 幻灯片：可开始/暂停自动播放，可配置播放间隔；播放时按顺序切换图片，退出后停止计时器。
- [ ] 全屏模式：可进入和退出全屏；全屏下图片、工具栏或悬浮控制不遮挡主要内容。
- [ ] EXIF 信息面板：可展示拍摄参数、分辨率、文件大小、创建时间；无 EXIF 的图片显示“无可用 EXIF”而不是空白错误。
- [ ] 快速信息：侧边栏或悬浮面板可在不离开查看器的情况下查看当前图片基本信息。
- [ ] 收藏图片：在查看器中切换收藏状态后，收藏页和图片列表状态同步更新。
- [ ] 复制图片路径：执行后系统剪贴板包含当前图片绝对路径，并给出成功或失败反馈。
- [ ] 移动到其他合集：从查看器移动图片后，原合集图片数减少，目标合集图片数增加，文件路径和数据库记录一致。
- [ ] 删除图片：支持移到回收站或直接删除；删除前有确认，删除后当前查看器自动切到合理的相邻图片或空状态。
- [ ] 查看所在文件夹：可打开系统文件管理器并定位到当前图片所在目录；路径不存在时提示原因。
- [ ] 快捷键：默认快捷键覆盖翻页、缩放、收藏、全屏、信息面板；快捷键冲突时有提示。

## 合集管理

- [ ] 导入合集：选择包含 500 张图片的文件夹后，系统自动扫描、入库、生成缩略图、提取元数据，并在合集列表出现该合集。
- [ ] 自动封面：新合集默认使用第一张可解码图片作为封面；用户手动指定封面后，重启应用仍保持自定义封面。
- [ ] 统计信息：合集图片数量和总文件大小与实际支持格式图片一致；不支持文件不计入统计。
- [ ] 合集名称：默认名称为文件夹名；编辑名称后列表、详情页、搜索结果同步更新。
- [ ] 合集描述：可新增、编辑、清空描述；保存后重启仍可见。
- [ ] 合集标签：可为合集绑定多个标签；在合集详情、列表卡片和标签筛选中一致显示。
- [ ] 合集评分：支持 0-5 星评分；评分可清空，按评分排序和筛选时结果正确。
- [ ] 合集收藏：可收藏/取消收藏合集；收藏页按收藏时间展示。
- [ ] 查看统计：打开合集详情会更新最后查看时间和查看次数；重复进入次数按规则递增。
- [ ] 列表/网格视图：合集可在列表视图和网格视图切换；切换后排序、筛选条件不丢失。
- [ ] 编辑合集：名称、描述、标签、评分、封面一次保存成功后，数据库和界面状态一致。
- [ ] 删除合集记录：删除合集记录时可选择是否删除物理文件夹；未选择删除物理文件夹时磁盘文件保留。
- [ ] 合并合集：多选合集合并到目标合集或新合集后，图片文件和数据库记录进入目标合集，源合集处理结果清晰可见。
- [ ] 拆分合集：按标签等条件拆分合集后，新合集包含匹配图片，原合集和新合集图片数可核对。
- [ ] 导出合集信息：可导出合集名称、路径、描述、标签、评分、图片数量、总大小等信息到文件；导出文件可重新打开检查内容。
- [ ] 排序：支持按名称、图片数量、总大小、创建时间、评分、最近查看、查看次数排序；每项均可升序/降序切换。
- [ ] 筛选：支持按标签、评分区间、日期范围、收藏状态、图片数量范围筛选；组合条件下结果满足全部条件。

## 图片管理

- [ ] 缩略图网格：进入合集详情后显示图片缩略图网格，缩略图大小可调整，自适应列数不会造成布局重叠。
- [ ] 虚拟滚动：万级图片合集滚动时只渲染可视区域附近项目，滚动过程无明显卡顿或白屏。
- [ ] 图片预览入口：单击图片显示预览或选中状态，双击进入查看器并定位到同一张图片。
- [ ] 图片信息：图片详情展示文件名、分辨率、文件大小、格式、创建时间、EXIF 信息；数据与磁盘文件一致。
- [ ] 单图打开：可从图片列表打开当前图片，查看器顺序与当前排序/筛选结果一致。
- [ ] 单图重命名：重命名后磁盘文件名、数据库记录、列表显示同步更新；同名冲突时阻止并提示。
- [ ] 单图移动：移动到其他合集后，文件实际移动，原合集和目标合集统计同步变化。
- [ ] 单图复制：复制到其他合集后，原文件保留，目标合集产生新记录，重复路径或文件名冲突有处理策略。
- [ ] 单图删除：删除前确认；删除后数据库记录、缩略图缓存和当前列表状态更新。
- [ ] 单图查看信息：右键或详情入口可打开信息面板，不需要进入全屏查看器。
- [ ] 批量移动：多选图片移动到目标合集后，所有选中项移动成功或给出逐项失败结果。
- [ ] 批量复制：多选图片复制到目标合集后，目标合集图片数增加，失败项可重试或查看原因。
- [ ] 批量删除：多选图片删除前显示数量和影响范围；删除后列表、数据库和文件系统一致。
- [ ] 批量打标签：多选图片添加/移除标签后，所有选中图片标签状态一致，标签筛选可命中。
- [ ] 批量评分：多选图片评分后，按评分排序和筛选结果正确。
- [ ] 右键菜单：图片右键菜单包含打开、重命名、移动、复制、删除、查看信息等操作；禁用状态符合当前选择。
- [ ] 拖拽操作：拖拽图片到其他合集可完成移动；拖拽过程中目标高亮，失败时原数据不变。
- [ ] 图片排序：支持按文件名、文件大小、分辨率、创建时间、评分、收藏状态排序；排序结果可用测试数据核对。
- [ ] 图片筛选：支持按格式、分辨率区间、文件大小区间、评分、收藏状态筛选；组合条件下结果满足全部条件。

## 标签

- [ ] 创建标签：可输入名称和选择颜色创建标签；名称为空或重复时有校验提示。
- [ ] 编辑标签：修改标签名称或颜色后，所有已关联合集和图片同步展示新值。
- [ ] 删除标签：删除前提示影响范围；删除后相关合集/图片关联被移除，其他数据不受影响。
- [ ] 标签重命名：重命名后搜索、筛选、标签云和详情页全部使用新名称。
- [ ] 标签颜色自定义：颜色选择后在标签列表、合集卡片、图片信息和筛选条件中一致显示。
- [ ] 合集多标签：一个合集可绑定多个标签，保存后可逐个移除。
- [ ] 图片多标签：一张图片可绑定多个标签，批量标签操作不覆盖未选择移除的已有标签。
- [ ] 标签筛选合集/图片：点击或选择标签后，合集和图片结果只包含对应标签数据。
- [ ] 标签云：标签云展示所有已使用标签，并能反映标签使用数量或权重；点击标签可触发筛选。

## 搜索

- [ ] 全局搜索合集名称：输入合集名称关键词后返回匹配合集，大小写或中英文输入按设计规则匹配。
- [ ] 全局搜索图片文件名：输入图片文件名关键词后返回匹配图片，并显示所属合集和路径信息。
- [ ] 全局搜索标签：输入标签关键词后返回匹配标签、带该标签的合集和图片。
- [ ] 高级格式搜索：可多选图片格式，结果只包含所选格式图片。
- [ ] 高级分辨率搜索：设置宽高或区间后，结果只包含满足分辨率条件的图片。
- [ ] 高级大小搜索：设置文件大小区间后，结果只包含满足大小条件的图片。
- [ ] 高级标签搜索：多选标签后，结果按约定的 AND/OR 规则准确返回。
- [ ] 高级评分搜索：评分区间可同时作用于合集和图片，结果不包含区间外数据。
- [ ] 高级日期搜索：日期范围可筛选创建时间或导入时间，边界日期包含规则明确且可验证。
- [ ] 高级收藏搜索：收藏状态筛选可区分已收藏和未收藏合集/图片。
- [ ] 组合条件搜索：格式、分辨率、大小、标签、评分、日期、收藏状态组合时，结果满足全部启用条件。
- [ ] 搜索结果分类：结果按合集/图片分类展示，数量统计与实际结果一致。
- [ ] 点击跳转定位：点击合集结果进入合集详情；点击图片结果进入所属合集并定位或打开对应图片。
- [ ] 搜索依赖：搜索结果必须来自已导入、已入库、已同步的数据库记录；文件监听或增量扫描后搜索结果随之更新。

## 重复检测

- [ ] SHA256 完全重复检测：对内容完全相同但文件名不同的图片分到同一重复组；内容不同的图片不被误判为完全重复。
- [ ] pHash 视觉相似检测：对尺寸或压缩不同但视觉近似的图片分到相似组，并提供可调整或可说明的相似阈值。
- [ ] 检测结果分组：重复/相似图片按组展示，每组至少显示文件路径、大小、分辨率和所属合集。
- [ ] 删除建议：系统给出保留建议，例如保留分辨率更高、文件更新、已收藏或评分更高的图片；建议原因可见。
- [ ] 批量删除重复图片：用户选择要删除的重复项后，可批量删除；保留项不会被删除。
- [ ] 保留指定图片：用户可手动指定每组保留项；执行后每组至少保留一张图片。
- [ ] 安全处理：删除前显示影响数量和路径；失败项保留原文件并显示原因。
- [ ] 检测依赖：重复检测只处理已入库且文件仍存在的图片；文件缺失时先标记异常或跳过并提示。

## 收藏/最近

- [ ] 收藏图片快速访问：收藏页可跨合集展示所有收藏图片，点击后打开图片或定位到所属合集。
- [ ] 收藏合集快速访问：收藏页可展示收藏合集，点击后进入合集详情。
- [ ] 收藏时间排序：收藏图片和收藏合集可按收藏时间排序，新增收藏出现在正确位置。
- [ ] 最近合集：最近查看的合集按查看时间展示，重复查看同一合集时更新位置而不是重复插入。
- [ ] 最近图片：最近查看的图片按查看时间展示，点击可重新打开或定位。
- [ ] 保留数量配置：修改最近浏览保留数量后，超出数量的历史记录按规则清理。
- [ ] 持久化：收藏和最近浏览数据在应用重启后仍保留。

## 设置

- [ ] 默认标签设置：配置导入默认标签后，新导入合集或图片自动带上对应标签，已有数据不被意外修改。
- [ ] 自动检测新文件夹：开启后已导入目录新增图片会自动同步；关闭后不会自动入库。
- [ ] 自动生成缩略图：开启后导入时后台生成缩略图；关闭后仅在需要显示时生成或显示占位。
- [ ] 缩略图大小：支持小/中/大/自定义；调整后合集详情和图片网格即时更新且布局稳定。
- [ ] 主题：支持亮色、暗色、跟随系统；切换后所有主要界面可读且重启后保持设置。
- [ ] 语言：支持中文/英文；切换后菜单、按钮、设置项和主要提示文本更新。
- [ ] 自定义快捷键：可修改快捷键绑定，冲突时阻止保存或提示覆盖策略。
- [ ] 默认快捷键查看：设置页可查看默认快捷键列表，内容与实际行为一致。
- [ ] 缩略图缓存大小限制：设置缓存上限后，缓存清理策略能把缓存控制在限制内或说明保留原因。
- [ ] 清理缓存：执行后缩略图缓存文件减少或清空，界面可按需重新生成缩略图。
- [ ] 缓存路径设置：修改缓存路径后，新缓存写入新位置，旧缓存处理策略明确。
- [ ] 数据库备份：可生成数据库备份文件，备份文件存在且大小合理。
- [ ] 数据库恢复：从备份恢复后，合集、图片、标签、收藏、最近浏览和设置与备份内容一致。
- [ ] 重建索引：执行重建索引后搜索、排序、筛选可正常工作，异常数据有报告。
- [ ] 导出数据：可导出合集、图片、标签、收藏、历史等数据；导出文件可被人工检查或再次导入验证。

## 文件监听

- [ ] 监听已导入合集目录：应用运行时修改已导入目录，系统能检测到文件新增、删除、重命名事件。
- [ ] 自动新增图片：向已导入目录放入支持格式图片后，数据库新增记录，缩略图按设置生成，合集统计更新。
- [ ] 自动删除图片：从磁盘删除已入库图片后，数据库状态更新，列表不再展示失效图片或展示明确缺失状态。
- [ ] 自动重命名图片：重命名图片文件后，数据库路径和文件名更新，标签、评分、收藏等元数据保留。
- [ ] 自动检测文件夹重命名：已导入合集文件夹重命名后，合集路径更新，原合集元数据保留。
- [ ] 监听开关：关闭自动检测后，文件系统变化不会自动修改数据库；手动扫描仍可同步。
- [ ] 事件去抖：批量复制、批量删除或软件生成缩略图时，不会因大量文件事件造成重复入库或界面卡死。

## 性能/稳定/兼容性

- [ ] 缩略图网格滚动：在万级图片合集下滚动接近 60fps，滚动过程中不出现持续白屏、错位或明显输入阻塞。
- [ ] 万级合集浏览：导入并打开万级图片合集后，列表、筛选、排序和进入查看器均可完成，应用内存保持在可接受范围内。
- [ ] 图片打开速度：已缓存图片打开小于 200ms，首次解码小于 1s；测试结果记录机器配置和图片规格。
- [ ] 导入性能：1000 张图片导入小于 30s；导入过程有进度、可取消或失败恢复机制。
- [ ] 大文件稳定性：打开超大分辨率或大体积图片时应用不崩溃，失败时显示可理解错误。
- [ ] 数据库损坏恢复：模拟数据库损坏时，应用能提示并提供恢复、备份或重建入口。
- [ ] 异常退出保护：导入、缩略图生成、批量移动/删除过程中异常退出，重启后数据库和文件状态可继续或修复。
- [ ] 简洁易用：主界面包含菜单栏、工具栏、侧边栏、主内容区；常用入口不用超过合理点击层级。
- [ ] 键盘快捷键：核心浏览、查看、收藏、搜索功能可用键盘完成，并与设置中的快捷键一致。
- [ ] 拖拽操作：支持拖拽图片到合集等计划内操作，并在无效目标上阻止执行。

## 用户场景

- [ ] 场景 1 导入新合集：选择包含 500 张图片的文件夹后，系统完成扫描、缩略图生成、元数据提取；用户可编辑合集名称、描述、标签、评分，完成后合集出现在列表中。
- [ ] 场景 2 浏览合集：用户从合集列表进入详情，看到缩略图网格，滚动浏览，双击图片进入查看器，使用左右键切换，并按 F 收藏当前图片。
- [ ] 场景 3 查找图片：用户输入图片文件名关键词，系统返回匹配图片；点击结果进入对应合集并定位到该图片。
- [ ] 场景 4 整理合集：用户多选合集执行合并，选择目标合集或创建新合集；系统移动图片并更新合集统计。
- [ ] 场景 5 清理重复：用户启动重复检测，系统扫描全库、展示重复图片组；用户选择删除项后批量删除并保留指定图片。
- [ ] 场景 6 按标签浏览：用户点击“风景”等标签后，系统筛选出所有带该标签的合集，并允许继续浏览结果。
