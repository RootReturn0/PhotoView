# PhotoView - Windows 图片查看器 & 合集管理工具

## 一、项目背景

下载了大量图片，按合集（文件夹）分类组织，但缺乏高效的管理和浏览手段。Windows 自带图片查看器功能有限，不支持 webp，且无法对合集进行增删改查管理。需要一个本地化的图片查看 + 合集管理一体化工具。

---

## 二、核心痛点

### 2.1 浏览体验痛点

| # | 痛点 | 说明 |
|---|------|------|
| 1 | **格式支持不全** | Windows 默认查看器不支持 webp、avif 等新格式，需额外安装编解码器 |
| 2 | **大图加载慢** | 高分辨率图片（4K+）打开慢，缩放卡顿 |
| 3 | **切换不流畅** | 文件夹内图片多时，上一张/下一张切换有延迟 |
| 4 | **缩略图生成慢** | 资源管理器缩略图缓存机制差，大文件夹加载缩略图极慢 |
| 5 | **无快速预览** | 无法在不打开图片的情况下快速浏览内容 |

### 2.2 合集管理痛点

| # | 痛点 | 说明 |
|---|------|------|
| 1 | **合集信息缺失** | 文件夹只有名称，无封面、描述、标签、评分等元数据 |
| 2 | **查找困难** | 合集多了以后，只能靠文件夹名搜索，无法按标签/评分/日期等多维度检索 |
| 3 | **合集操作繁琐** | 合并、拆分、移动合集需要手动操作文件系统，容易出错 |
| 4 | **重复图片** | 同一张图片可能存在于多个合集中，占用空间且难以发现 |
| 5 | **合集状态不清** | 不知道哪些合集已看/未看/收藏，缺乏状态标记 |
| 6 | **排序方式单一** | 只能按文件系统排序，无法按评分/图片数/创建时间等自定义排序 |

### 2.3 图片管理痛点

| # | 痛点 | 说明 |
|---|------|------|
| 1 | **无法标记** | 无法对单张图片进行收藏、评分、标记 |
| 2 | **无EXIF查看** | 无法快速查看图片的拍摄信息、分辨率、文件大小等 |
| 3 | **批量操作难** | 无法批量移动、删除、重命名图片 |
| 4 | **跨合集查找** | 无法在所有合集中搜索某类图片 |

## 三、功能需求

### 3.1 图片查看器

#### 3.1.1 格式支持
- 支持主流格式：jpg/jpeg, png, gif, bmp, ico, tiff
- 支持新格式：webp, avif
- 支持矢量格式：svg

#### 3.1.2 查看功能
- **缩放**：适应窗口 / 实际大小 / 自由缩放（滚轮/快捷键）
- **旋转**：90° 旋转
- **翻页**：键盘左右 / 鼠标侧键 / 点击按钮
- **幻灯片**：自动播放，可配置间隔时间
- **全屏模式**：沉浸式浏览

#### 3.1.3 图片信息
- EXIF 信息面板：拍摄参数、分辨率、文件大小、创建时间
- 快速查看：侧边栏或悬浮面板显示

#### 3.1.4 快捷操作
- 收藏图片
- 复制图片路径
- 移动到其他合集
- 删除图片（移到回收站或直接删除）
- 查看图片所在文件夹

### 3.2 合集管理

#### 3.2.1 合集导入
- 选择文件夹导入为合集
- 自动扫描文件夹内所有支持的图片格式
- 自动生成合集封面（第一张图或用户指定）
- 自动统计图片数量和总大小

#### 3.2.2 合集信息
- 名称（默认为文件夹名，可修改）
- 封面图片（可自定义）
- 描述
- 标签（多标签）
- 评分（0-5 星）
- 收藏状态
- 图片数量
- 总文件大小
- 创建时间
- 最后查看时间
- 查看次数

#### 3.2.3 合集操作
- **创建**：导入新文件夹
- **查看**：列表视图 / 网格视图
- **编辑**：修改名称、描述、标签、评分、封面
- **删除**：删除合集记录（可选是否删除物理文件夹）
- **合并**：多个合集合并为一个
- **拆分**：按条件拆分合集（如按标签）
- **导出**：导出合集信息为文件

#### 3.2.4 合集排序
- 按名称排序
- 按图片数量排序
- 按总大小排序
- 按创建时间排序
- 按评分排序
- 按最近查看排序
- 按查看次数排序
- 升序/降序切换

#### 3.2.5 合集筛选
- 按标签筛选
- 按评分区间筛选
- 按日期范围筛选
- 按收藏状态筛选
- 按图片数量范围筛选

### 3.3 图片管理（合集内）

#### 3.3.1 图片浏览
- 缩略图网格视图
- 自适应列数（可调整缩略图大小）
- 快速滚动加载（虚拟滚动）
- 单击预览，双击进入查看器

#### 3.3.2 图片信息
- 文件名
- 分辨率
- 文件大小
- 格式
- 创建时间
- EXIF 信息（如有）

#### 3.3.3 图片操作
- **单图操作**：打开、重命名、移动到其他合集、复制到其他合集、删除、查看信息
- **批量操作**：批量移动、批量复制、批量删除、批量打标签、批量评分
- **拖拽操作**：拖拽图片到其他合集移动

#### 3.3.4 图片排序
- 按文件名排序
- 按文件大小排序
- 按分辨率排序
- 按创建时间排序
- 按评分排序
- 按收藏状态排序

#### 3.3.5 图片筛选
- 按格式筛选
- 按分辨率区间筛选
- 按文件大小区间筛选
- 按评分筛选
- 按收藏状态筛选

### 3.4 标签系统

#### 3.4.1 标签管理
- 创建标签（名称 + 颜色）
- 编辑标签
- 删除标签
- 标签重命名
- 标签颜色自定义

#### 3.4.2 标签使用
- 为合集打标签（多标签）
- 为图片打标签（多标签）
- 按标签筛选合集/图片
- 标签云展示

### 3.5 搜索功能

#### 3.5.1 全局搜索
- 搜索合集名称
- 搜索图片文件名
- 搜索标签

#### 3.5.2 高级搜索
- 组合条件搜索：
  - 格式（多选）
  - 分辨率区间
  - 文件大小区间
  - 标签（多选）
  - 评分区间
  - 日期范围
  - 收藏状态

#### 3.5.3 搜索结果
- 分类展示（合集 / 图片）
- 点击跳转到对应位置

### 3.6 重复检测

#### 3.6.1 检测方式
- 完全相同检测：文件哈希（MD5/SHA256）
- 视觉相似检测：感知哈希（pHash）

#### 3.6.2 检测结果
- 分组展示重复/相似图片
- 显示文件路径、大小、分辨率
- 提供删除建议

#### 3.6.3 去重操作
- 批量删除重复图片
- 保留指定图片

### 3.7 收藏夹

- 跨合集的收藏图片快速访问
- 收藏的合集快速访问
- 收藏时间排序

### 3.8 最近浏览

- 最近查看的合集
- 最近查看的图片
- 可配置保留数量

### 3.9 设置

#### 3.9.1 导入设置
- 默认标签设置
- 是否自动检测新文件夹
- 是否自动生成缩略图

#### 3.9.2 显示设置
- 缩略图大小（小/中/大/自定义）
- 主题（亮色/暗色/跟随系统）
- 语言（中文/英文）

#### 3.9.3 快捷键设置
- 自定义快捷键绑定
- 查看默认快捷键

#### 3.9.4 缓存管理
- 缩略图缓存大小限制
- 清理缓存
- 缓存路径设置

#### 3.9.5 数据管理
- 数据库备份
- 数据库恢复
- 重建索引
- 导出数据

### 3.10 文件系统监听

- 监听已导入合集文件夹的变化
- 自动检测新增图片并入库
- 自动检测删除图片并更新数据库
- 自动检测文件夹重命名并更新

## 四、非功能需求

### 4.1 性能要求
- 缩略图网格滚动流畅（60fps）
- 万级图片合集流畅浏览
- 图片打开速度 < 200ms（已缓存）/ < 1s（首次解码）
- 合集导入速度：1000 张图片 < 30s

### 4.2 易用性要求
- 界面简洁直观
- 支持键盘快捷键
- 支持拖拽操作
- 支持右键菜单
- 操作可撤销

### 4.3 稳定性要求
- 大文件处理不崩溃
- 数据库损坏可恢复
- 异常退出数据不丢失

### 4.4 兼容性要求
- 支持 Windows 10/11
- 支持高 DPI 显示
- 支持多显示器

## 五、用户场景

### 场景 1：导入新合集
用户下载了一个包含 500 张图片的文件夹，想导入管理：
1. 点击"导入合集"按钮
2. 选择文件夹
3. 系统自动扫描图片、生成缩略图、提取元数据
4. 用户编辑合集名称、添加描述、打标签、评分
5. 导入完成，合集出现在列表中

### 场景 2：浏览合集
用户想浏览某个合集的图片：
1. 在合集列表中点击合集
2. 进入合集详情，看到缩略图网格
3. 滚动浏览缩略图
4. 双击某张图片进入查看器
5. 使用左右键切换图片
6. 按 F 收藏当前图片

### 场景 3：查找图片
用户记得某张图片在某个合集，但不记得具体位置：
1. 在搜索框输入图片文件名关键词
2. 系统返回匹配的图片列表
3. 点击图片跳转到对应合集并定位

### 场景 4：整理合集
用户想把多个合集合并：
1. 在合集列表中多选要合并的合集
2. 右键选择"合并"
3. 选择目标合集或创建新合集
4. 系统将所有图片移动到目标合集

### 场景 5：清理重复
用户想清理重复图片释放空间：
1. 点击"重复检测"功能
2. 系统扫描全库
3. 展示重复图片组
4. 用户选择要删除的图片
5. 批量删除

### 场景 6：按标签浏览
用户想看所有"风景"标签的合集：
1. 在侧边栏点击"风景"标签
2. 系统筛选出所有带"风景"标签的合集
3. 用户浏览筛选结果
